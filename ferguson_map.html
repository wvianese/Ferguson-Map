Ferguson Housing Analysis Tool

About: This map shows vacancy and abandonment risk predictions for 8,877 properties in Ferguson, MO using machine learning and AI image analysis.

How to use:

  • Layers: Toggle different prediction models and risk factors using the menu (bottom-right)
  • Click properties: Click any colored parcel to see detailed scores and property information
  • Hover: Hover over parcels to see addresses - the colored square shows the current layer's color
  • Search: Type an address to see matching results, then click a result or press Enter to zoom to it
  • Legend: Click "Color Guide" (bottom-left) to show/hide the legend

Models:

  • 1) Statistical Model: Uses parcel features such as building value and ownership type in a regularized logistic regression model to estimate the probability that a property is abandoned. The model is trained on the confirmed abandoned list so each feature is weighted by how strongly it relates to known outcomes.
  • 2) AI Image Analysis: Uses Google Street View imagery and a vision classifier to estimate probabilities for occupied, abandoned, and vacant-lot conditions from visible property condition. This adds what the property physically looks like, not just what is in county records.
  • 3) Combined Score: Blends both models with validation-tuned weights (70% statistical, 30% AI image). This weighted average is used as the primary score because it is more stable and more accurate than using either source by itself.

Accuracy vs Confirmed Abandoned List:

  • Statistical Model: 93.65%
  • AI Image Analysis: 86.89%
  • Combined Score: 95.39%

Data Sources:

  • St. Louis County Open Data (Property Records)
  • Ferguson City Property Restoration Program (Confirmed List)
  • Google Street View Imagery
